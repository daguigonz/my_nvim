# My nvim 

## 📌 Install Nvim

```bash
$ sudo npm install
```

## 📌 Ingresa a la carpeta 

```bash
~/.config/
```
## 📌 Copiar directorio de proyecto a las siguiente ruta (Mac).

```bash
~/.config/nvim
```

## 📌 Instal server autocompletado 

```bash
npm install -g typescript typescript-language-server
```

## 📌 Ejecutar 

```bash
$ nvim init.vim
```

```bash
 esc 
 :PlugInstall
```

## 📌 Ayudas 

```bash
,, = 'auto completar html, js y css'
```
```bash
f2 = open treeview
```

## Mas información
